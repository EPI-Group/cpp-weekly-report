# 两两交换链表中的节点

## 题目

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

  

**你不能只是单纯的改变节点内部的值**，而是需要实际的进行节点交换。

  

 

  

**示例:**

  

```
给定 1->2->3->4, 你应该返回 2->1->4->3.
```

## 代码

struct ListNode* swapPairs(struct ListNode* head){  

​        struct ListNode **p=&head,*a,*b;  

​        while((a=*p)&&(b=a->next))   

​	{  

​            a->next=b->next;  

​            b->next=a;  

​            *p=b;  

​            p=&(a->next);  

​        }  

​        return head;  

 }
