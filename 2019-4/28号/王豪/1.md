# 合并两个有序链表

## 题目

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

  

**示例：**

  

```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

## 代码



//先将两个链表合并，后面再排序

struct ListNode* mergeTwoLists(struct ListNode* l1, struct ListNode* l2){  

​    int a=0;  

​    if(!l1)   

​        return l2;  

​    if(!l2)  

​        return l1;  

​    if(l1||l2)  

​    {  

​        struct ListNode *p=l1;  

​        struct ListNode *q= NULL;  

​        while(p->next)  

​        {  

​            p=p->next;   

​        }  

​        p->next=l2;          //合并两个链表  

​        p=l1;  

​        while(p->next)  

​        {  

​            q=p->next;  

​            while(q)  

​            {  

​                if(p->val>q->val)  

​                {  

​                    a=p->val;  

​                    p->val=q->val;  

​                    q->val=a;  

​                } 

​                q=q->next;  

​            }  

​            p=p->next;  

​        }  

​    }  

​    return l1;  

}
