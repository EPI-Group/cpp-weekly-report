# 2的幂

## 题目

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

  

**示例 1:**

  

```
输入: 1
输出: true
解释: 20 = 1
```

  

**示例 2:**

  

```
输入: 16
输出: true
解释: 24 = 16
```

  

**示例 3:**

  

```
输入: 218
输出: false
```

## 代码

bool isPowerOfTwo(int n) {  

​    if(n<=0)               //如果a为零或负，肯定不是2的倍数  

​        return false;  

​    while(n!=1)  

​    {  

​        if(n%2==0)         //让n一直除以2，如果一直除到1的话就说明它是2的幂，否则不是  

​            n=n/2;  

​        else  

​            return false;  

​    }  

​    return true;  



}
