给定范围 [m, n]，其中 0 <= m <= n <= 2147483647，返回此范围内所有数字的按位与（包含 m, n 两端点）。

示例 1: 

输入: [5,7]
输出: 4
示例 2:

输入: [0,1]
输出: 0



```c
int rangeBitwiseAnd(int m, int n) {
        // 高位不变，低位变。故最终值由高位确定。
        // how to decide 高位的位数？
        // 分析：从左向右看，第一位不同的地方记为i，包括这一位之后的所有位必然有机会取到0和1。 因为区间一定会包含10000000这样的数，
        // 故后面取&的结果一定位0, 故最终结果为： i之前的位值不变，i之后的位值均为0
    int count = 0;
    while(n!=m){
        n = (n>>1);
        m = (m>>1);
        count++;
    }
    while(count>0){
        n = (n<<1);
        m = (m<<1);
        count--;
    }
    return n;
}
```
