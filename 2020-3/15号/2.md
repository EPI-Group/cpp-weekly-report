# 最大子序和

## 题目

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

示例:

输入: [-2,1,-3,4,-1,2,1,-5,4],
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

## 代码

```c
int maxSubArray(int* nums, int numsSize) 
{
    int sum = nums[0];
    int now = nums[0];
    
    for(int i = 1; i < numsSize; i++)
    {
        if(now < 0)
            now = nums[i];
        else
            now = now + nums[i];
        
        if(now > sum)
            sum = now;
    }
    return sum;
}
```
